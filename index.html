Git

환경조회

git status
파일 상태보기
git log
commit history 보기

Git Setting

git init
지금부터 이 폴더에서 작업을 시작하곘다
git add [파일명][.]
이 파일들을 버전관리 하겠다[tracked]
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
사용자가 누구인가 기록한다.

삭제

git reset [파일명]
Stage Area에서 working Storage로 이동

복사하기

git clone[url]
원격 저장소에 있는 파일을 복제한다.
git pull [url]
원격에 있는 파일의 내용과 로컬에 있는 내용이 자동으로 추가된다.
git fetch [url]
원격지에 있는 자료를 가져온다.

복사본 만들기

git branch 브랜치명
새로운 브랜치로 들어가 폴더생성
git checkout 브랜치명
브랜치 폴더로 들어간다.
git switch 브랜치명
브랜치 폴더로 들어간다.
git merge 브랜치명
현재 브랜치에서 지정한 브랜치 작업이 merge 된다.
git branch -D 브랜치명
작업이 끝난 브랜치 삭제한다.
